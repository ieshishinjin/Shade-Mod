# Shade Mod GUI 说明

> 适用于 Shade Mod v0.0.4+

---

## 快捷键

| 快捷键 | 功能 | 可自定义 |
|--------|------|----------|
| **R** | 打开剧情菜单 | ✅ 设置→按键绑定→Shade 模组 |
| **U** | 打开 AI 控制面板 | ✅ 设置→按键绑定→Shade 模组 |
| **↑ / ↓** | 选择剧本（剧情菜单内） | — |
| **Enter** | 确认选择（剧情菜单内） | — |
| **1~9** | 快速选择对话选项 | — |
| **G** | 打开画廊（剧情菜单内） | — |
| **点击** | 跳过打字机效果 / 继续对话 | — |
| **ESC** | 关闭当前界面 | — |

按键绑定可在 Minecraft 设置 → 按键绑定 → "Shade 模组" 分类中修改。

## 剧情菜单（StoryMenuScreen）

按 **R** 打开，列出所有可用剧本：

```
┌──────────────────────────────────────┐
│            ✦ 剧情选择                 │
│                                      │
│  ○ 第一章：苏醒                      │   ◀ 剧本列表
│  ▶ 第二章：林中迷途 (进行中)         │     点击选中
│  √ 序章                              │
│  ○ 第三章                              │
│                                      │
│  选中: 第一章：苏醒                   │
│  ○ 未开始                            │  ◀ 右侧详情面板
│  ID: chapter1_wake_up                │
│  你在一个陌生的房间里醒来...          │
│                                      │
│  [ 画廊 ] [ 关闭 ]     [ ▶ 开始 ]    │
└──────────────────────────────────────┘
```

### 操作

| 操作 | 方式 |
|------|------|
| 选择剧本 | 点击列表项，或按 **↑ / ↓** |
| 开始/继续 | 选中后点击 **「▶ 开始 / 继续」**，或按 **Enter** |
| 打开画廊 | 点击 **「画廊」**，或按 **G** |
| 关闭 | 点击 **「关闭」**，或按 **ESC** |

### 剧本状态标记

| 标记 | 含义 |
|------|------|
| **○** | 未开始 |
| **▶** | 进行中（当前活跃） |
| **√** | 已完成 |

## 对话框（StoryDialogScreen）

剧情播放时自动弹出，Galgame 风格对话框：

```
┌────────────────────────────────────────────┐
│                                            │
│    ┌────────┐  §e§l阿卡娅                    │
│    │ 80×80  │                              │
│    │  头像   │  「你终于醒了。感觉怎么样？」 │   ← 对话白色
│    │         │  一个温和的声音从旁边传来...    │   ← 旁白灰色
│    └────────┘                              │
│                                            │
│    ┌──────────────────────────────┐        │
│    │ [1] 「这是哪里？」            │        │   ← 选项按钮
│    ├──────────────────────────────┤        │
│    │ [2] （保持沉默）              │        │
│    └──────────────────────────────┘        │
│                          §7[ 点击继续 ]      │
└────────────────────────────────────────────┘
```

### 打字机效果

- 文字逐字显示（每 tick 2 个字）
- 点击或按任意键立即显示全文
- 对话用 `「」` 包裹显示为白色，旁白/描述显示为灰色

### 对话框类型

| type | 类型 | 行为 |
|------|------|------|
| 1 | 对话 + 选项 | 文字显示完后显示选项按钮 |
| 2 | 仅旁白 | 无说话人头衔，灰色文字 |
| 3 | 关闭 | 关闭当前对话框 |
| 4 | 结束语 | 淡入→显示→点击淡出后关闭 |

### 操作

| 操作 | 方式 |
|------|------|
| 跳过打字机效果 | 点击或按任意键 |
| 继续下一句 | 文字显示完后点击，或按任意键（选项模式除外） |
| 选择选项 | 点击选项按钮，或按 **1~9** 数字键 |
| 关闭结束语 | 点击，淡出后自动关闭 |
| 退出（结束语） | **ESC** 关闭 |

### 头像

- 支持通过 `ResourceLocation` 路径加载自定义贴图
- 无贴图时显示默认占位框
- 尺寸：80×80 像素

## Quest 追踪 HUD（StoryQuestOverlay）

有活跃 Quest 时，在游戏画面**右上角**显示：

```
    ┌─ Quest 名称 ──────────────────┬─
    │ §6✦ 逃离森林                   │
    │ §7○ 抵达河边 §e[ 3§7/§e5 §7]    │
    │ §a✔ 收集木材 §a[ 5§7/§a5 §7]    │
    │                        │
    └────────────────────────┘
```

### 显示内容

| 元素 | 说明 |
|------|------|
| Quest 名称 | 金色显示在顶部 |
| 目标列表 | 每个目标一行，带状态标记 |
| 进度条 | `当前值/最大值` 格式 |
| 右侧竖线 | 金色装饰线 |

### 状态标记

| 标记 | 含义 |
|------|------|
| **○** | 未完成 |
| **✔** | 已完成 |

## AI 控制面板（AiControlScreen）

按 **U** 打开，浮窗式界面，游戏画面背景可见：

```
┌──────────────────────────────────┐
│ ✦ AI 剧情生成                    │
│ ───────────                      │
│ ● 剧情进度                       │
│    ○ 第一章：苏醒  (进行中)       │
│ ───────────                      │
│ ● AI 引擎                        │
│ [DeepSeek] [Ollama] [启用] [禁用]│
│ [智谱AI(免费)] [讯飞星火(免费)]   │
│ [Groq]           [更多]          │
│ ───────────                      │
│ ● 生成参数                       │
│ [温度0.6] [温度0.8] [温度1.0]    │
│ [短] [长]                        │
│ ───────────                      │
│ ● 生成剧情                       │
│ ┌────────────────────────┐       │
│ │ 输入剧情描述...          │       │
│ └────────────────────────┘       │
│       [▶ 生成剧情]               │
│ ───────────                      │
│ [测试连接] [状态] [设置] [关闭]   │
│                      AI Studio   │
└──────────────────────────────────┘
```

### 按钮功能

| 按钮 | 作用 |
|------|------|
| **DeepSeek** | 切换到 DeepSeek AI 引擎 |
| **Ollama** | 切换到 Ollama 本地引擎 |
| **启用** | 开启 AI 生成功能 |
| **禁用** | 关闭 AI 生成功能 |
| **智谱AI(免费)** | 一键填入智谱 AI 配置 |
| **讯飞星火(免费)** | 一键填入讯飞星火配置 |
| **Groq** | 一键填入 Groq 配置 |
| **更多** | 查看所有推荐服务商 |
| **温度0.6 / 0.8 / 1.0** | 调整生成随机性 |
| **短 / 长** | 调整生成文本长度 |
| **▶ 生成剧情** | 用输入框中的描述生成剧情 |
| **测试连接** | 测试当前 AI 连接 |
| **状态** | 查看当前配置信息 |
| **设置** | 打开设置二级页面 |
| **关闭** | 关闭控制面板 |

### 文本输入

- 直接键盘输入剧情描述
- **Enter** 快速生成
- **ESC** 关闭
- 最多 300 个字符

## AI 设置页（AiSettingsScreen）

在控制面板点击 **「设置」** 打开：

```
┌──────────────────────────────────┐
│  AI 设置                     关闭│
│ ───────────                      │
│ ● 选择模型                       │
│ [DeepSeek] [Ollama] [自定义]     │
│                                  │
│ ● API 密钥                       │
│ ┌────────────────────────┐[保存] │
│ │ sk-...                 │       │
│ └────────────────────────┘       │
│ 当前: DeepSeek                   │
│                                  │
│       [← 返回主菜单]             │
│                      AI Studio   │
└──────────────────────────────────┘
```

### 模型说明

| 模型 | 说明 |
|------|------|
| **DeepSeek** | 推荐，需 API Key |
| **Ollama** | 本地运行，免费，无需密钥 |
| **自定义** | 任意 OpenAI 兼容 API |

### 自定义模式

选中 **「自定义」** 后展开额外字段：

```
┌──────────────────────────────────┐
│ ● 选择模型                       │
│ [DeepSeek] [Ollama] [自定义]     │
│                                  │
│ ● API 端点                       │
│ ┌────────────────────────────┐   │
│ │ https://api.example.com/   │   │
│ └────────────────────────────┘   │
│                                  │
│ ● 模型名                         │
│ ┌────────────────────────────┐   │
│ │ gpt-4o-mini                │   │
│ └────────────────────────────┘   │
│                                  │
│ ● API 密钥                       │
│ ┌────────────────────────┐[保存] │
│ │ xxx                   │       │
│ └────────────────────────┘       │
│ 当前: 自定义                     │
│                                  │
│       [← 返回主菜单]             │
└──────────────────────────────────┘
```

### 操作

| 操作 | 方式 |
|------|------|
| 切换模型 | 点击 **DeepSeek / Ollama / 自定义** |
| 保存配置 | 点击 **「保存」** 或按 **Enter** |
| 切换输入字段（自定义） | 按 **Enter** 跳到下一字段 |
| 返回主菜单 | 点击 **「← 返回主菜单」** |
| 关闭 | 点击右上角 **「关闭」** 或按 **ESC** |
